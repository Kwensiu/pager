## 第一步：启动应用程序

### 安装 Pager

1. 从 [GitHub Releases](https://github.com/Kwensiu/Pager/releases) 下载最新版本的安装包
2. 运行安装程序并按照提示完成安装
3. 启动 Pager 应用程序

### 首次启动

启动后，您将看到：

- **左侧侧边栏**：用于管理和组织您的网站
- **右侧主窗口**：用于浏览和访问网站
- **默认示例**：侧边栏中包含一些示例分组和网站，帮助您快速了解功能

---

## 第二步：添加您的第一个网站

### 添加网站

1. 点击侧边栏底部的 **"+"** 按钮
2. 在弹出的菜单中选择 **"添加网站"**
3. 在对话框中填写以下信息：
   - **网站名称**：例如 "ChatGPT"
   - **网站 URL**：例如 "https://chat.openai.com"
   - **描述**（可选）：添加备注信息
4. 点击 **"保存"** 按钮

### 自动获取图标

Pager 会自动从网站 URL 获取 favicon 图标。如果图标未显示，您可以：

1. 右键点击网站
2. 选择 **"编辑"**
3. 点击图标旁边的刷新按钮

---

## 第三步：组织您的网站

### 创建分类

分类是 Pager 的一级组织结构，用于将不同类型的网站分组。

1. 右键点击侧边栏空白处
2. 选择 **"添加分类"**
3. 输入分类名称（如"工作"、"娱乐"、"学习"）
4. 点击 **"保存"**

### 创建分组

分组是分类下的二级组织结构，用于进一步细分网站。

1. 右键点击已创建的分类
2. 选择 **"添加分组"**
3. 输入分组名称（如"AI工具"、"开发工具"）
4. 点击 **"保存"**

### 移动网站到分组

有两种方式将网站移动到分组：

**方式一：拖拽**

- 直接拖拽网站到目标分组

**方式二：右键菜单**

1. 右键点击网站
2. 选择 **"移动到"**
3. 选择目标分组

---

## 第四步：浏览网站

### 打开网站

1. 点击侧边栏中的网站
2. 网站将在主窗口的新标签页中打开

### 使用导航工具栏

标签页顶部提供导航工具栏，包含以下功能：

- **后退**：返回上一个页面
- **前进**：前进到下一个页面
- **刷新**：重新加载当前页面
- **主页**：返回网站首页
- **地址栏**：显示和编辑当前 URL

### 管理标签页

- **新建标签页**：点击标签页栏右侧的 "+" 按钮
- **关闭标签页**：点击标签页上的 "×" 按钮
- **切换标签页**：点击标签页标题

## 第五步：基本设置

### 打开设置

点击右上角的 **设置图标**（齿轮图标）打开设置对话框。

### 外观设置

- **主题模式**：
  - 浅色主题：明亮界面
  - 深色主题：暗色界面，减少眼睛疲劳
  - 跟随系统：自动匹配系统主题

- **界面语言**：
  - 中文：简体中文界面
  - 英文：英语界面

- **侧边栏显示模式**：
  - 显示所有网站：折叠时显示所有网站图标
  - 仅显示展开的分组：折叠时只显示展开状态的分组

### 浏览器设置

- **启用 JavaScript**：控制是否允许网页执行 JavaScript
- **允许弹出窗口**：控制是否允许网页打开新窗口
- **保存会话数据**：保存登录状态和浏览历史
- **退出时清除缓存**：关闭应用时清理临时文件

## 常用操作

### 添加多个网站

1. 重复"添加网站"步骤
2. 或者使用批量导入功能（未来版本支持）

### 编辑网站信息

1. 右键点击网站
2. 选择 **"编辑"**
3. 修改名称、URL 或刷新图标
4. 点击 **"保存"**

### 删除网站或分组

1. 右键点击项目
2. 选择 **"删除"**
3. 确认删除操作

### 搜索网站

1. 使用侧边栏顶部的搜索框
2. 输入网站名称或 URL 的一部分
3. 搜索结果将实时显示

### 重命名分类或分组

1. 右键点击分类或分组
2. 选择 **"重命名"**
3. 输入新名称
4. 点击 **"保存"**

## 快捷键参考

### 导航快捷键

| 快捷键                   | 功能               |
| ------------------------ | ------------------ |
| `Ctrl/Cmd + T`           | 新建标签页         |
| `Ctrl/Cmd + W`           | 关闭当前标签页     |
| `Ctrl/Cmd + Tab`         | 切换到下一个标签页 |
| `Ctrl/Cmd + Shift + Tab` | 切换到上一个标签页 |

### 侧边栏快捷键

| 快捷键                | 功能                 |
| --------------------- | -------------------- |
| `Alt/Cmd + Shift + S` | 显示/隐藏侧边栏      |
| `Alt/Cmd + 数字键`    | 切换到对应位置的网站 |

### 窗口快捷键

| 快捷键                | 功能         |
| --------------------- | ------------ |
| `Alt/Cmd + Shift + T` | 切换窗口置顶 |
| `Alt/Cmd + Shift + M` | 切换小窗模式 |
| `F11`                 | 全屏模式     |

## 实用技巧

### 技巧1：快速添加常用网站

1. 将浏览器书签导出为 HTML 文件
2. 未来版本将支持书签导入功能

### 技巧2：使用分组提高效率

- **按用途分组**：工作、娱乐、学习
- **按项目分组**：项目A、项目B
- **按频率分组**：每天使用、偶尔使用

### 技巧3：利用侧边栏折叠

- 点击侧边栏顶部的折叠按钮
- 折叠后只显示网站图标
- 适合小屏幕或多任务场景

### 技巧4：备份您的数据

1. 定期导出配置（设置 → 数据管理 → 导出配置）
2. 备份用户数据目录
3. 使用数据同步功能

### 技巧5：使用拖拽排序

- **分类排序**：拖拽分类标题调整顺序
- **分组排序**：在分类内拖拽分组调整顺序
- **网站排序**：在分组内拖拽网站调整顺序
- **跨分组移动**：将网站拖拽到其他分组

## 常见问题

### Q: 如何恢复默认设置？

A: 在设置页面中，点击"数据管理" → "重置为默认设置"。

### Q: 网站图标不显示怎么办？

A: 尝试右键点击网站 → "编辑" → "刷新图标"，或检查网络连接。

### Q: 如何导出我的配置？

A: 在设置页面中，点击"数据管理" → "导出配置"。

### Q: 支持哪些扩展格式？

A: Pager 支持标准的 Chrome 扩展格式，包括文件夹、ZIP 和 CRX 文件。
