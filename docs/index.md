## 📖 文档概览

### Pager 是什么？

Pager 是一个基于 Electron + React + TypeScript 构建的桌面应用程序，提供侧边栏界面来管理和访问多个网站，旨在通过结构化组织网络资源来提高生产力。

### 主要特性

- 🎯 **侧边栏管理** - 两级组织结构（分类和分组）管理网站
- 🔧 **扩展支持** - 支持标准 Chrome 扩展
- 🎨 **主题定制** - 浅色/深色主题，跟随系统
- 🌍 **国际化** - 完整的中文/英文支持
- ⚡ **性能优化** - 自动内存清理和会话隔离
- 🔒 **隐私保护** - 浏览器指纹伪装和权限管理
